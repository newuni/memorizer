# Security Policy

Thanks for helping keep memorizer safe.

## Supported versions

We currently support security fixes for the latest release on `main`.

## Reporting a vulnerability

Please **do not** open public issues for security vulnerabilities.

Instead, report privately via:
- GitHub Security Advisories (preferred): `Security` tab in this repo
- Or direct contact: open a private message to maintainer (@newuni)

Include:
- Affected version/commit
- Reproduction steps or PoC
- Impact assessment
- Suggested mitigation (if any)

## Response targets

- Initial acknowledgment: within 72 hours
- Triage decision: within 7 days
- Fix/release timing: depends on severity, but we prioritize critical issues first
